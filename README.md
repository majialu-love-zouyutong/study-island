# 学习岛——无人自习室O2O平台

## web端商家管理系统

📋 后台首页（仪表盘 Dashboard）
├── 今日预约数、总订单数、收入
├── 热门时段图表 / 使用率

📍 门店与座位管理
├── 门店信息设置（店名、地址、营业时间等）
├── 座位管理（新增/编辑/状态开关）

📆 预约与订单管理
├── 预约记录（用户 + 座位 + 时间 + 状态）
├── 订单列表（编号 + 金额 + 状态 + 时间）
├── 退款审核（可选）

🧑 用户管理
├── 用户列表（昵称、联系方式）
├── 用户预约记录（跳转查看）
├── 黑名单管理（可选）

💡 设备管理（物联网）
├── 门锁状态（在线/离线）
├── 空调、灯光开关状态
├── 远程控制（可选）

📊 数据统计
├── 收入折线图（日/周/月）
├── 使用热度分析（时段柱状图）
├── 座位利用率排行（可选）

🔧 系统设置
├── 预约时间段配置
├── 营业时间/是否暂停营业
├── 座位预约规则（最多预约时长等）

📢 公告管理
├── 发布新公告
├── 公告列表与编辑
